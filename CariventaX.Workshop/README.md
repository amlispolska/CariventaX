CariventaX.Workshop

This folder contains the Workshop feature app. It depends on CariventaX.Base.

Recommended structure:
- table/
- page/
- codeunit/
- report/
- Translations/

Move only workshop-specific files here and keep shared objects in CariventaX.Base.

Build order: publish CariventaX.Base first, then CariventaX.Workshop.
