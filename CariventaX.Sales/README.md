CariventaX.Sales

This folder contains the Sales feature app. It depends on CariventaX.Base.

Recommended structure:
- table/
- page/
- codeunit/
- report/
- Translations/

Move only sales-specific files here and keep shared objects in CariventaX.Base.

Build order: publish CariventaX.Base first, then CariventaX.Sales.
